# Défi Fouille de Texte - 2009

## Objectif

Ce projet s'inscrit dans le cours d'*Apprentissage Supervisé* de M. [Loic Grobol](https://github.com/LoicGrobol). Il vise à répondre à la tâche n°3 du DEFT-2009. Le but étant d'entrainer un classifieur pour catégoriser automatiquement des discours politiques en fonction du parti politique de l'orateur.
